UGATIT玩耍指南

发表于 2019-09-07 | 分类于 教程
字数统计: 1.7k | 阅读时长 ≈ 5

Introduce

UGATIT是韩国游戏公司NCSOFT开源的一个AI图像风格转换项目，其主要功能是将真人照片转为漫画风格，换句话说就是可以通过技术手段实现照片的“次元转换”，废话不多说，上图~

右侧是真人原图（图片资源源于网络，侵删），左侧是由UGATIT自动转化生成的，可不是P站大触画的哦~

如你所见，UGATIT可以实现二次元与三次元的双向转化：如果你输入任意小姐姐的自拍，就可以解锁一个二次元妹子；如果你想看一个二次元萌妹子突破次元壁的样子，那么就给模型输入一张二次元画像，然后你就会看到小姐姐在三次元的样子啦~此外，这个AI模型还可以实现不同动物间的转换、照片与油画间的转换等功能

怎么样，想必各位绅士已经感叹技术の伟大了吧~

使用方法：

1.自行训练

将源码git或者download到本地，解压后在根目录下建立dataset文件夹，并在此文件夹内建立selfie2anime子文件夹，这个子文件夹是存放训练数据与测试数据用的，当然名字也可以取其他的。

然后在selfie2anime中建立testA,testB,trainA,trainB四个文件夹，分别用来存放训练与测试数据，其中“A”、“B”分别用来存放不同风格的数据，比如可以在文件A中放三次元图片，在B文件夹中放二次元图片。

至于数据来源，你可以用爬虫爬，或者自己手动下载，图片格式、大小、名称都不限。

但是这样自己搞数据还是太慢了，不过好在有大佬已经将自己的数据集分享到网上了，我们直接下载来用就好，这里还要给大佬掌声~

大佬的数据集文件是按照上述文件夹格式整理好的，因此直接解压到工程根目录下就好啦再次感谢大佬

测试的过程相比训练而言快了很多，基本取决于testA与testB中图片的数量。

怎么样，结果是不是很令人振奋？

不过你也许会看到一些辣眼睛的图片，没错，这个模型的转换成功率不是100%，有一定概率失败。成功率主要取绝于图片数据本身的一些特性。经过笔者初步研究发现，清晰度越高，脸部暴露面积越大，面部特征越明显的图片成功的概率越大，至于深层次原因，还尚未研究。

如果你想改进这个模型的效果，那就大胆fork吧，成功后记得喊我去给你点小星星~

Analysis

笔者对图像领域知之甚少，不敢妄加分析，特此找了一些资料供大家共同学习，节选如下：

UGATIT是图翻图的翻译算法，采用无监督方法训练，不需要数据对，这点就很神奇了~

从结构上，UGATIT由两个GAN组成的。

一个GAN，负责把妹子的自拍变成二次元小姐姐。这是从源领域到目标领域的翻译。

另一个GAN，负责把二次元小姐姐再变回三次元自拍。这是从目标领域到源领域的翻译。

这样，就有两套Generator与Discriminator的组合。
